# 🎮 Hand Gesture Controlled Tic-Tac-Toe

A Computer Vision powered Tic-Tac-Toe game where players use hand gestures and pinch detection to play against an AI opponent.

## 🚀 Features

- Real-time hand tracking
- Gesture-based cell selection
- Pinch gesture click detection
- AI opponent using Minimax Algorithm
- Easy, Medium, Hard modes
- Scoreboard system
- Winning line animation
- Reset functionality

## 🛠️ Tech Stack

- Python
- OpenCV
- MediaPipe
- cvzone
- NumPy

## 🎯 Controls

- Pinch → Select Cell
- 1 → Easy Mode
- 2 → Medium Mode
- 3 → Hard Mode
- R → Reset Board
- ESC → Exit
